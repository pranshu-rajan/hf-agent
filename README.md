# 🤖 HF Agents Course Work

This repository contains my implementations, experiments, and learnings from the **Hugging Face Agents Course**.  
It focuses on building intelligent AI agents using Large Language Models (LLMs), tool-calling, and real-world automation workflows.

---

## 📌 Overview

AI Agents are the next evolution in Generative AI — systems that can **reason, plan, and act using tools**.

In this project, I explored:
- Agent-based architectures
- Tool calling using LLMs
- Multi-step reasoning workflows
- Integration of AI models into applications

---

## ⚙️ Features

- 🤖 Intelligent AI Agent implementation  
- 🧠 LLM-based reasoning system  
- 🔧 Custom tool integration  
- 📂 File reading & processing tools  
- 🎤 Speech-to-text capability  
- 📺 YouTube transcription tool  
- 🖼️ Image description tool
